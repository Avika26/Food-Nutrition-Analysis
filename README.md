A comprehensive \*\*from-scratch\*\* machine learning project analyzing USDA nutrition data.



\---



\## 📋 Project Objective



Analyze \*\*335 food items\*\* using three machine learning algorithms built \*\*entirely from scratch\*\* (NumPy \& Pandas only) — Linear Regression, KNN, and K-Means.



\---



\## ✨ Key Features



\- Data cleaning and preprocessing from raw CSV

\- Linear Regression for calorie prediction

\- KNN for Healthy vs Unhealthy classification

\- K-Means for unsupervised food clustering

\- Handled severe class imbalance using random oversampling (from scratch)



\---



\## 📊 Results



\### KNN Classification



| Model                  | Accuracy | Precision | Recall | F1 Score |

|------------------------|----------|-----------|--------|----------|

| KNN (Imbalanced)       | 1.0000   | 1.0000    | 1.0000 | 1.0000   |

| \*\*KNN (Balanced)\*\*     | \*\*0.9897\*\* | \*\*0.9756\*\* | \*\*1.0000\*\* | \*\*0.9877\*\* |



\- \*\*Linear Regression\*\*: \*\*R² Score = 0.8529\*\*

\- \*\*K-Means\*\*: 3 meaningful clusters (Low Calorie, Protein-Rich, High-Carb)



\---



\## 🧠 Major Learnings



\- Class imbalance can create misleading 100% accuracy

\- Proper imbalance handling dramatically improves real-world model performance

\- Building ML algorithms from scratch builds strong fundamentals
